命令行模式(按 Esc 生效)
F
: 查找并且替换
P
: 打开命令配置
R
: 清除代码块格式
A
: 在上面插入代码块
B
: 在下面插入代码块
Shift-V
: 粘贴到上面
V
: 粘贴到下面
Z
: 撤销删除
Shift-M
: 合并选中单元格, 如果只有一个单元格被选中
L
: 切换行号
O
: 选择单元格的输出
Q
: 关闭页面
Shift-空格
: 向上滚动
空格
: 向下滚动






编辑模式(按 Enter 生效)
Ctrl-D
: 删除整行
Ctrl-Shift-F
: 打开命令配置
Ctrl-Shift-P
: 打开命令配置
Shift-Enter
: 运行代码块, 选择下面的代码块
Ctrl-Enter
: 运行选中的代码块
Alt-Enter
: 运行代码块并且插入下面
Ctrl-Shift-Minus
: 在鼠标处分割代码块
